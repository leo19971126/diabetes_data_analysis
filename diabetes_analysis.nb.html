Diabetes Analysis

Introduction

The objective of this project is to use different data analysis tools to explore this diabetes dataset that originates from NIDDK (National Institute of Diabetes and Digestive and Kidney Diseases)

library(moderndive)
library(ggplot2)
library(dplyr)

    filter, lag

    intersect, setdiff, setequal, union
library(infer)

##Read data

diabetes_raw <- read.csv('diabetes.csv')
head(diabetes_raw)

##Clean data Here I cleaned the data by getting rid of the zeros for variables such as glucose, blood pressure, etc. The raw dataset have 768 observations, and after cleaning the data theres 392 observations left, which should still be enough for data analysis.

clean_diabetes <- diabetes_raw %>% 
  filter(Glucose != 0 & BloodPressure != 0 & SkinThickness != 0 & Insulin != 0 & BMI != 0 )

##Data visualization #Glucose VS Diabetes Patients with diabetes are correlated with higher blood glucose level. So let us see if this is true for our dataset:


ggplot(data = clean_diabetes, aes(x = Outcome, y = Glucose, group = cut_width(Outcome, 1)))+ geom_boxplot()

#Glucose VS BMI :


ggplot(data = clean_diabetes, aes(x = BMI, y = Glucose, color=Outcome))+ geom_point()
